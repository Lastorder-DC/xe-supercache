사용 가능한 설정 문법은 아래와 같습니다. 필요에 따라 변경하시기 바랍니다.

  • 파일 캐시: file
    소규모 사이트나 Memcached를 사용하기 곤란한 웹호스팅 환경에서 쓸 수 있는 방법입니다.
    files/cache/store 폴더가 디스크 용량을 많이 차지할 수 있으니 주의하십시오.
  • Memcached: memcache://127.0.0.1:11211
    대규모 사이트에서 최적의 선택입니다.
    다른 서버에서 구동할 경우 IP 주소와 포트를 변경하십시오.
  • APC: apc
    불가피한 경우 외에는 권장하지 않습니다. PHP 5.5 이상 버전에서는 사용하지 마십시오.
    특히 캐시 용량이 가득차면 안정성이 떨어진다는 제보가 있으니, 용량을 넉넉하게 설정하시기 바랍니다.
  • WinCache: wincache
    윈도우 서버 전용이며, 불가피한 경우 외에는 권장하지 않습니다.

You can use the following syntax. Please adjust to your needs.

  • Files: file
    Suitable for small sites on shared server where Memcached is not available.
    Caution: This can cause a large amount of files to be stored in the files/cache/store folder.
  • Memcached: memcache://127.0.0.1:11211
    Best choice for large sites.
    Please change the IP address and port if Memcached is running on a different server.
  • APC: apc
    Not recommended, especially in PHP 5.5 or higher.
    Caution: APC is known to become unstable when the cache size is too small.
  • WinCache: wincache
    Not recommended unless there is no other choice. Only for Windows servers.
